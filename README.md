# Text to QR Code Generator

A simple web application that allows users to generate QR codes from text or URLs. Users can select the size of the QR code, preview it instantly, and download it as an image.

---

## Features

- Generate QR codes from any text or URL
- Select QR code size: 100x100, 150x150, 200x200, 300x300
- QR code preview is centered and responsive
- Download QR code as PNG
- Clean, professional UI using HTML, CSS, and JavaScript

---

## Demo

1. Enter text or URL in the input box.
2. Choose a QR code size from the dropdown.
3. Click **Generate QR** to see it on the right side.
4. Click **Download QR** to save the generated QR image.

---

## Technologies Used

- HTML5
- CSS3 (Flexbox for layout and centering)
- Vanilla JavaScript
- [QR Code API](https://goqr.me/api/) for generating QR images
